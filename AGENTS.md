## Agent skills

### Issue tracker

Issues and PRDs are tracked in GitHub Issues for `lyydsheep/Copyability`. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage uses the default five-label state vocabulary. See `docs/agents/triage-labels.md`.

### Domain docs

This repository uses a single-context domain documentation layout. See `docs/agents/domain.md`.

### Validation

Run `npm run typecheck`, `npm test`, and `node --check copyability.user.js` before publishing changes.
