# Userscript as a prototype carrier

The user understands that Tampermonkey manages and injects a userscript while the userscript supplies the page behavior, and has chosen this lightweight delivery mechanism for the first Copyability prototype. Future lessons can assume this distinction and focus on userscript behavior rather than re-teaching the delivery model.

## Evidence

After the introductory lesson, the user explicitly accepted Tampermonkey as the first-version carrier for Copyability.
